# 2734 围巾

## 题目描述

你的小店今天开始营业，由于是新开张，所以第一天 24 小时营业。小店专门卖围巾，你有 $$n$$ 条围巾。已知第一天 $$m$$ 个顾客的信息。第 $$i$$ 个顾客的到店的信息格式如下：

第一行：一个整数 $$k$$，

接下来有 $$k$$ 行，每行有 3 个整数，即：

> $$t_1$$ $$c_1$$ $$p_1$$（表示第 $$i$$ 位顾客有 $$p_1\%$$ 概率在第 $$t_1$$ 点到店，且愿意花 $$c_1$$ 元买 1 条围巾）
>
> $$t_2$$ $$c_2$$ $$p_2$$（表示第 $$i$$ 位顾客有 $$p_2\%$$ 概率在第 $$t_2$$ 点到店，且愿意花 $$c_2$$ 元买 1 条围巾）
>
> $$t_3$$ $$c_3$$ $$p_3$$（表示第 $$i$$ 位顾客有 $$p_3\%$$ 概率在第 $$t_3$$ 点到店，且愿意花 $$c_3$$ 元买 1 条围巾）
>
> ……
>
> $$t_k$$ $$c_k$$ $$p_k$$（表示第 $$i$$ 位顾客有 $$p_k\%$$ 概率在第 $$t_k$$ 点到店，且愿意花 $$c_k$$ 元买 1 条围巾）

一个顾客不会多次到店，由于是第一天营业，你也不想那么忙碌，所以每一个小时最多只能有 1 位顾客到店，而且你每次最多只卖 1 条围巾，所以 1 个顾客最多只能买到 1 条围巾。

每当顾客到店时，你可以拒绝卖毛巾给他，也可以卖 1 条围巾给他。由于知道了 $$m$$ 个顾客的信息，所以你希望用最优的策略使得第一天能收到的钱的期望值最大，输出最大的期望值。

## 输入格式

多组测试数据。

第一行，一个整数 $$r$$。$$1 \leq g \leq 5$$。

每组测试数据格式如下：

第一行，两个正整数 $$n$$ 和 $$m$$。

接下来是 $$m$$ 位顾客的信息，每位顾客信息的格式如题目所描述。

## 输出格式

$$r$$ 行，每行一个实数。你的误差范围不能超过 $$10^{-6}$$。

## 样例解释

第一组测试数据解释：

只有 1 条围巾要卖。

最优策略：第 1 个顾客就算 8 点到店，也不卖给他。12 点时，第 2 位顾客一定会到达。

$$80\% \times 10 + 11\% \times 100 + 9\% \times 0 = 19.0$$

## 数据范围

$$1 \leq n \leq 24,\,1 \leq m \leq 24$$；

$$0 \leq t_1 < t_2 < t_3 < \cdots < t_k \leq 23$$，$$1 \leq c_1,\,c_2,\,c_3,\,\cdots,\,c_k \leq 100$$，$$1 \leq p_1,\,p_2,\,p_3,\,\cdots,\,p_k \leq 100$$。

数据保证：$$p_1 + p_2 + p_3 + \cdots + p_k < 100$$，即第 $$i$$ 位顾客有 $$100 - (p_1 + p_2 + p_3 + \cdots + p_k)$$ 的概率不到店。对于任意时间 $$0 \leq t \leq 23$$，$$t$$ 最多只出现在一个客户的信息中。
